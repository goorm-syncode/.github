# Sync Code

코딩 학습 플랫폼을 개발하는 팀입니다.  
웹 환경에서 학습과 개발 경험을 결합한 서비스를 만드는 것을 목표로 합니다.

---

## 🚀 Learn Code

**Learn Code**는 Sync Code 팀이 개발하는 코딩 학습 플랫폼입니다.  
웹 기반 IDE 환경에서 문제 풀이와 학습 흐름 관리를 제공합니다.

### 주요 기능

- Mission 기반 문제 풀이
- 웹 IDE 환경에서 코드 작성 및 실행
- 학습 흐름 관리 (진행 상태, 이어하기)
- 실시간 채팅 기능

현재는 MVP 단계로,  
프론트엔드와 백엔드의 기본 구조를 안정적으로 구축하는 것을 목표로 하고 있습니다.

---

## 📦 저장소

### Frontend
[web-ide-frontend](https://github.com/goorm-syncode/web-ide-frontend)

**Frontend Application**
- Vite + React 기반 프론트엔드
- Mission 목록 및 IDE 화면 제공
- 채팅 UI 및 사용자 인터랙션 처리

---

### Backend
[web-ide-backend](https://github.com/goorm-syncode/web-ide-backend)

**Backend Server**
- Spring Boot 기반 API 서버
- Mission, 사용자 진행 상태, 채팅 등 도메인 처리
- 코드 실행 및 채점 시스템 연동
